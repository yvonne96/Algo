Step by Step Example


Let us take the array of numbers "4 7 1 3",

First Pass:

( 4 7 1 3) -> ( 4 7 1 3 ), since 4 < 7, nothing changes

( 4 7 1 3 ) -> ( 4 1 7 3 ), since 7 > 1 swap

( 4 1 7 3 ) -> ( 4 1 3 7 ), since 7 > 3 swap

Second Pass:

( 4 1 3 7 ) -> ( 1 4 3 7), since 4 > 1 swap

( 1 4 3 7 ) -> ( 1 3 4 7 ), since 4 > 3

list is now sorted but bubble sort will keep running until there as no swaps which will be the third pass.
