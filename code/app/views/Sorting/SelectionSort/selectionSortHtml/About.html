Selection Sort


Selection sort is an in-place sorting algorithm. It sorts the array by having two variables i and j. All elements to the left of array[i] are already sorted and everything to the right isn't.

For every element (i), all the elements to right are looked at (j) and if array[j] is smaller than the previous minimum value seen (min_found) than that becomes the new min_found. Once every element to the right has been looked at array[i] is swapped with min_found.
