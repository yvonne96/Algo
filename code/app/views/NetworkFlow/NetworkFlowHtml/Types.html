Types of Network Flow Problems


Max Flow A start vertex (s) is given, a goal vertex (t) is given and the network of vertices and edges with flow capacties are given. The goal is to maximise the flow from vertex (s) to vertex (t).
Minimum Cost A start vertex (s) is given, a goal vertex (t) is given, the network of vertices and edges with flow capacties are given. Each edge also has a cost associated with it. The goal is to minimise the cost while maximising the flow from vertex (s) to vertex (t).
Bipartite Matching A set of edges are provided. The goal is to find a maximum matching of the edges so that no two edges share the same endpoint.
