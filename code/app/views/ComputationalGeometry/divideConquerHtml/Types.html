Types of Computational Geometry Problems


Static Definition
Input is given & the corresponding output needs to be constructed or found

Example
Convex Hull : Given a set of points find the smallest shape that contains the points on the outside
line segment intersection : Find all the intersections of a group of line segments
Geometric Query Definition
Input has a search space and a query. These vary over the problem's instances

Example
Range Searching : Process's a set of points and counts the number of points inside a certain region
Nearest Neighbour : Process's a set of points and determines which point is closest to a certain point
Dynamic Definition
The main aim is to find an efficient algorithm that will find a solution repeatedly after each modification of the input data

Example
Dynamic Convex Hull
Variations Definition
Problems may be categorized as belonging to one or more of these groups

Example
Point in polygon : Deciding whether its inside or outside a given polygon.
